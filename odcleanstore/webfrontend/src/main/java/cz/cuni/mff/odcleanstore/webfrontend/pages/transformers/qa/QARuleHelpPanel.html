QA rule

Each rule consists of a GroupGraphPattern1 filter, quality decrease coefficient and description, as described in the following table.

FilterCoefficientDescription
GroupGraphPattern [GROUP BY ... [HAVING ...]]x in [0, 1]description
Example
{{?s anatomy:limbs ?o} FILTER (?o > 4)}0.4To many limbs

Any snippet of SPARQL2 to which "SELECT * FROM ... WHERE" can be prepended is a valid filter and describes a property of a named graph that the author of the rule finds defective.
