The backend components of ODCleanStore. This artifact contains transformers for processing of incoming graphs and components for queries over the clean database.

The interface of all transformers is defined in package {@link cz.cuni.mff.odcleanstore.transformer transformer}. Several transformers are integrated in ODCleanStore:

  • {@link cz.cuni.mff.odcleanstore.qualityassessment Quality Assessment} - computes score of incoming named graphs based on user-defined rules
  • {@link cz.cuni.mff.odcleanstore.linker Object Identification & Linkage} - interlinks resources which represent the same real-world entity.

Components for queries over the clean database include:

  • {@link cz.cuni.mff.odcleanstore.queryexecution Query Execution} - retrives triples & metadata relevant for a given query from the clean database
  • {@link cz.cuni.mff.odcleanstore.conflictresolution Conflict Resolution} - resolves conflict in data from multiple (possibly conflicting) sources and adds quality of the result and provenance information
